## 安全边界

| 边界类型 | 可强制 | 无法强制 | 缓解措施 |
|----------|--------|----------|----------|
| 本地操作 | 通过AVM包装入口 | 直接文件修改 | 提交时检查 |
| Git Hooks | pre-commit/commit-msg/pre-push | --no-verify | 远程Ruleset兜底 |
| GitHub Ruleset | PR必需、状态检查、禁止直推 | 管理员关闭Ruleset | 审计日志 |
| CI检查 | 必需状态检查 | 手动跳过 | Ruleset required_status_checks |
| 审批签名 | HMAC绑定文件哈希 | 本地文件篡改 | 发布前重新验证 |
